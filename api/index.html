📡 API · Portfolio DAW

API pública RESTful con datos del portfolio de Sergio Daniel Martínez Gómez. Todos los endpoints devuelven JSON con CORS abierto.

GET/api/projects.jsonCatálogo completo
Listado de los 20 proyectos del portfolio con campos bilingües ES/EN, tecnologías, enlaces y metadatos.
fetch('https://daw1bsergiomg26.github.io/portfolio-daw-sergio/api/projects.json')
.then(r => r.json())
.then(projects => console.log(projects.length))
GET/api/blog.jsonEntradas del blog
Artículos técnicos del blog con contenido ES/EN, categorías, etiquetas y metadatos.
fetch('https://daw1bsergiomg26.github.io/portfolio-daw-sergio/api/blog.json')
.then(r => r.json())
.then(posts => console.log(posts))
GET/api/cv.jsonCurrículum
Datos profesionales: formación, skills, idiomas, soft skills y enlaces.
fetch('https://daw1bsergiomg26.github.io/portfolio-daw-sergio/api/cv.json')
.then(r => r.json())
.then(cv => console.log(cv))

🔧 Uso

Todas las rutas devuelven Content-Type: application/json con cabecera Access-Control-Allow-Origin: *. No se requiere autenticación. Los datos se actualizan con cada release del portfolio.
